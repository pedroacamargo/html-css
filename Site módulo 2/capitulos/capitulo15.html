Capítulo 15 - id e class

Aula 01 - id e class

  1. ID

    Deveremos colocar o id em alguma tag, por exemplo <h1 id="principal">, e assim, poderemos no arquivo.css colocar h1#principal {} e iremos alterar somente as tags com o id.

  2. CLASS

    Deveremos colocar o class em alguma tag, por exemplo <h1 class="ola", e assim, poderemos no arquivo.css colocar h1.ola {} e iremos alterar somente as tags com o id.

Aula 02 - Pseudo-classes

Ao utilizarmos uma tag <div>, no arquivo .css podemos dar uma pseudo-classe para esta div.
Exemplo:

                
    <style>
        .voltar {
            display: inline-block;
            background-color: lightgray;
            text-align: center;
            height: 200px;
            width: 200px;
            border; 1px solid black;
        }

        .voltar:hover {
            background-color: yellow;
        }
    </style>

    <div class="voltar">
        <p>Teste</p>
    </div>
:hover → Passar o mouse (hover your mouse)
