    1. <d1> → Lista de definições número 1.
    2. <dt> → É o termo, a palavra que irá ser definida/descrita/explicada.
    3. <dd> → É a definição do termo.
  • Exemplo:

    • HTML
    Linguagem de marcação para a criação de um site.
    CSS
    Linguagem de marcação para a criação do design de um site.
    JavaScript
    Linguagem de programação para a criação de interatividade de um site.
    Representação do código: 
    
<d1>

    <dd>Linguagem de marcação para a criação de um site.</dd>
    <dt>CSS</dt>
    <dd>Linguagem de marcação para a criação de um site.</dd>
    <dt>JavaScript</dt>
    <dd>Linguagem de programação para a criação de interatividade de um site.</dd>
</d1>

Aula 11 - Trabalhando com links

  1. Básico dos links
    • <a> </a> → Deverá escrever uma frase dentro dessas tags, de forma a criar uma âncora (hiperligação).
    • <a href="LINK" target="_blank" rel="external">
    • Target → _self _blank
    • rel → "external" "next" "prev" "nofollow"
    • A tag rel serve para facilitar a busca no google
  2. Usando links externos

    3. Visite o meu Instagram
    APARÊNCIA DO CÓDIGO:
    Visite o meu <a>href="https://www.instagram.com/pedroa_camargo/" target="_blank" rel="external">Instagram</a>

  3. Usando links internos

    4. Acesse a página 02 do nosso site.
    APARÊNCIA DO CÓDIGO:
    Acesse a <a href="pagaula11.html">página 02</a> do nosso site.
    OBS: é preciso ter um folder ou file no arquivo para que o arquivo na href funcione.
    → Além disso, caso o arquivo esteja dentro de uma pasta, deverá escrever antes ../ de forma a indicar que está contida na pasta anterior. (Função Linux). → ../resumo.html

  4. Tipos de rel:

    5. → external - Te leva para um site exterior ao teu.
    → next - Leva para um site próprio, para a próxima página do site.
    → prev - Leva para a página anterior, do seu site.
    → nofollow - Leva para um site especial, patrocinado, onde o público é limitado.

  5. Links para download

    6. Exemplo:
    Clique aqui para baixar o ZIP de um exemplo de um livro.

    Clique aqui para baixar o PDF de um exemplo de um livro.
    Representação do código feito acima: 

                
    < href="ex010/livro/meulivro.zip" download="meulivro.zip" type="application/zip">Clique aqui para baixar o ZIP de um exemplo de um livro.</a>
    <a href="ex010/livro/meulivro.pdf" download="meulivro.zip" type="application/pdf" target="blank">Clique aqui para baixar o PDF de um exemplo de um livro.</a>

    Depois exportamos as imagens para uma pasta criada.

  2. Imagens dinâmicas
    • Tags utilizadas:
      → <picture>
      → <source:media:type> --- Ao digitar isso, o código será criado automaticamente e terá a seguite aparencia: 
                      
<source media="(min-width: )" srcset="" type="image/">
    • min-width="" → colocamos o número de pixels (é mais fácil trocar para max-width)
    • srcset="" → colocamos o caminho até o png, pdf. Exemplo: imagens/foto-p.png
    • type="" → colocamos ao lado da barra o tipo de arquivo. Exemplo: /pdf /png
    • Exemplo de site com a imagem dinâmica. (Junto com seu código)

  3. Inserindo áudios no seu site.
    • Cuidados a tomar: Direitos autorais e tamanho do arquivo
    • Tags:
      <audio src=""> </audio> → Em "src" colocamos a source do arquivo (o destino).
      ⇒ Existem outras tags adicionais para colocarmos dentro da tag <audio>:
      • autoplay → Começa o áudio automaticamente quando entramos no site
      • controls → Aparece os controles para dar play e pausar
      • loop → O áudio repete-se
      <audio preload=""> </audio> → preload="metadata, auto, none" ← PASSAR O MOUSE EM CIMA
    • Problemas de compatibilidade
      Alguns navegadores não reconhecem alguns arquivos de áudio, forçando aos desenvolvedores de sites criarem vários tipos de arquivos de audio disponíveis num site.
      ⇒ Exemplo:
      • audio.MP3
      • audio.WAV
      • audio.OGG
      Devemos escolher alguns desses formatos de áudio de forma a haver mais compatibilidade num site e, além disso, há uma tag para adicionarmos vários tipos de áudio num site, caso um não funcione outro irá funcionar, e caso nenhum funcione, um link no fim será disponibilizado para fazer o download do audio.
      Exemplo de código: 
                          
<audio preload="metadata" controls autoplay loop>
    <source src="midia/musiquinhabraba.mp3" type="audio/mpeg"> 
    <source src="midia/musiquinhabraba.ogg" type="audio/ogg">
    <source src="midia/musiquinhabraba.wav" type="audio/wav">
    <p>Infelizmente, seu navegador não consegue reproduzir áudio. <a href="midia/musiquinhabraba.mp3"<CLIQUE AQUI PARA BAIXAR O ARQUIVO MP3</a><p>
</audio>
    • Devemos evitar arquivos .WAV, são muito pesados!

Aula 14 - Início CSS

  1. Tags utilizadas
    • background-color;→ Cor do fundo (página ou algo escrito.)
    • font-family→ Escolher o tipo de fonte.
    • font-size→ Escolher o tamanho da fonte (Em pixels)
    • color→ Cor da fonte ou da tag selecionada
    • text-align→ Muda o alinhamento do texto (Justify para ser o alinhamento total na página)
  2. Tipos de CSS
    • CSS inline, inserida na tag → Não é muito recomendada!
      Exemplo:
      <h1 style="color: red">
    • CSS no próprio arquivo .html
      Exemplo: 
                          
body {
    color: red;
    background-color: black;
    font-size: 20px
}
    • CSS em outro arquivo HTML
      → Neste caso, devemos criar uma file .CSS e no arquivo index.html ou qualquer outro digitar link:css
      Isso ficará assim:
      <link rel="stylesheet" href="style.css">
      No href devemos colocar a file em que o código CSS está.

      Além disso, devemos no começo do código CSS escrever @charset "UTF-8";
